制約別使用可能アルゴリズム

N <= 10～20
　・O(N!)→順列全探索(N<= 10まで)
　・O(2^N)→bit全探索(間に合わなければ再帰関数で枝狩りなど)
　・O(N^4)など→多重for文による全探索

N <= 100
　・O(N^3)→3重for文全探索

N <= 1000
　・O(N^2)→二重for文全探索
　・O(N^2 logN)→for文内でsort、二分探索(クエリ問題など)

N <= 2*10^5
　・O(NlogN)→sortからの二分探索

N <= 10^7
　・O(N)→線形探索(累積和、尺取りなど)

それ以上(N >= 10^7)
　・O(logN)→二分探索(sort済み)
　・O(1)→数学

STLまとめ

min,max
　・min(a,b),max(a,b) //O(1)
　・min({a,b,...,y,z}),max({a,b,...,y,z})
　・*min_element(v.begin(),v.end()),*max_element(v.begin(),v.end()) //O(N)

vector
　・push_back(x),pop_back()　//O(1)
　・sort(v.begin(),v.end())←昇順 //O(NlogN)
　・sort(v.rbegin(),v.rend())←降順
　・sort(v.begin(),v.end(),[](int a, int b) {return a < b})←ラムダ式による比較関数の変更(例はデフォルト)
　・reverse(v.begin(),v.end())
　・v.erase(unique(v.begin(), v.end()), v.end())←重複削除 //O(N)
　(全体の重複を消す場合→sort必須、隣り合った重複を消したい場合→sort必要なし)
　・it = v.erase(it)←次の要素を指すイテレーターが返る //後ろに詰める→O(N)

string
　・string S
　・S.substr(l,n)←Sのl文字目からn文字の部分文字列
　・stoi(S),stoll(S)←文字列を数字に変換
　・to_string(n)←数字を文字列に変換

stack //全てO(1)
　・stack<int> st
　・st.push(x), st.pop()
　・st.top()←一番上の要素を参照
　・st.empty()←空ならtrueを返す

queue //全てO(1)
　使用例：BFS
　・queue<int> que
　・que.push(x), que.pop()
　・que.front()←一番前の要素を参照
　・que.empty()←空ならtrueを返す

priority_queue
　使用例：ダイクストラ法、貪欲
　・priority_queue pq←大きい順
　・priority_queue<int, vector<int>, greater<int>> pq←小さい順
　・pq.push(x), pq.pop() //O(logN)
　・pq.top()←一番前の要素を参照 //O(1)
　・pq.empty()←空ならtrueを返す

deque //全てO(1)
　使用例：0-1BFS
　・deque dq
　・dq.front(), dq.back()
　・dq.push_front(x)
　・dq.push_back(x)
　・dq.pop_front()
　・dq.pop_back();

map //全てO(logN)
　・map<番地,アクセス先>
　以下使用例↓
　・map<string,int> mp
　・mp["sakuya"] = 398←要素へのアクセス、キーが存在しなければ自動追加
　・mp.count("keine")←存在すれば1、存在しなければ0を返す

set //全てO(logN)
　・set<int> s
　・s.insert(x)
　・s.erase(x)
　・auto it = s.lower_bound(x)←最初のx以上のイテレーター
　・if (it != s.end()) cout << *it←x以上の最小のものを出力

pair
　・pair<int,int> p = {a,b}
　・p.first,p.second←それぞれにアクセス

tuple
　・tuple<int,int,...,int> t←定義の例
　・get<i>(t)←i番目の要素にアクセス(0-index)
　・make_tuple(a,b,c,...,z)

binary_search //O(logN)
　＊昇順sortされている必要あり＊
　・binary_search(v.begin(),v.end(),x)←存在確認

lower_bound(以上),upper_bound(より大きい) //O(logN)
　＊昇順sortされている必要あり＊
　使用例：二分探索
　・int target = n←int型の場合の例、あらかじめ変数に入れておくと分かりやすい
　・auto it = lower_bound(v.begin(),v.end(),n)←イテレーター取得
　・if (it != v.end()) int index = it - v.begin()←インデックス取得

count
　・count(v.begin(),v.end(),x) //O(N)

find //O(N)
　・auto it = find(v.begin(),v.end(),x)←最初にxが見つかった時のイテレーターを返す
　・if (it != v.end()) int index = it - v.begin()←xが見つかったならインデックスを取得
　map,setの場合↓
　・auto it = mp.find(x)
　・if (it != mp.end())←存在する

next_permutation //O(N)
　使用例：順列全探索
　・昇順sortされている必要あり
　・next_permutation(v.begin(),v.end())←do-while文で使うことがほとんど

__builtin_popcount
　・__builtin_popcount(x)←xのうち1のビット数を返す
　・__builtin_popcountll(x)←long longに対応
